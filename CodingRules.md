# OVCT 專案命名規則

* 適用語法：Python

## 各項命名規範
### 使用之命名種類
* Pascal Case
  ```
  UserName
  ```
* Snake Case
  ```
  user_name or User_Name
  ```
### 各項命名規範
  |種類      						|命名規則  |範例 |注意事項|
  |-------------------------------|---------|-----|-------|
  |Folder  			    	|Snake    |shared_functions |  |
  |File  		  	   		|Pascal   |VpnFilter.py     |  |
  |Class & Function  	|Snake    |Select_One       |  |
  |Variable 		 	   	|Snake    |ovpn_hostname    |  |

(注意事項可以寫他的組成方式 e.g. 使用 V. + N. 命名 啥的)
